### **Cloud deployment in AWS:**1. **Set up VPC (subnets, Internet Gateway, NAT Gateway, route tables)**2. **Containerize the app with Docker and build the image**3. **Push the image to ECR**4. **Create an ECS cluster + task definition (tells ECS which image to run, how much CPU/RAM)**5. **Run the task as a Fargate service (ECS pulls from ECR and runs it)**6. **Add a Load Balancer in the public subnet pointing to your ECS service**7. **Configure Security Groups on the load balancer and the container**8. **(Optional but practical) Add a custom domain via Route 53 + SSL certificate via ACM**
